After a user starts working on an issue and a session is linked to it, show a "Resume Session" button on the issue card. Clicking it should reopen the associated claude code session, making it easy to pick up where you left off.

When a user clicks and ideate, plan, implement action on an issue, automatically rename the active Claude terminal session to reflect the issue being worked on (e.g., include the issue number and title). This gives clear context about what each session is for.
